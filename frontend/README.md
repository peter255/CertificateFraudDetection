# Frontend

React + TypeScript + Vite client for Certificate Fraud Detection.

See the [root README](../README.md) and [docs/GettingStarted.md](../docs/GettingStarted.md) for setup.

Quick local (without Docker):

```bash
cp .env.example .env
npm install
npm run dev
```

The preferred path for the full stack is Docker Compose from the repository root.
